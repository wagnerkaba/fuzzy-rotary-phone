## Tecnologias utilizadas:

- Java 11
- Spring Boot
- Swagger
- PostgreSQL
- Lombok
- Hibernate validator
- Spring Boot Actuator
- Spring Data JPA
- H2 Database para realizar testes unitários
- Mockito e AssertJ para testes unitários
